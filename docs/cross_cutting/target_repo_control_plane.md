# Target Repo Control Plane

`.specops/` documents the procedure, contracts, templates, schemas, and evaluation rules by which information becomes accepted specification truth.

It is not the specification itself.

## Destinations

```text
Draft/scratch:
  .specops/runs/

Process rules:
  .specops/process.md
  .specops/commands/
  .specops/templates/
  .specops/schemas/
  .specops/evals/

Reviewed provenance:
  docs/research/refinery/

Accepted truth:
  docs/CANON.md
  docs/decisions/
  docs/subsystems/
  docs/cross_cutting/
  docs/interfaces/
  docs/versions/
  docs/implementation/
```
