# Non-goals

## v0 non-goals

- Hosted SaaS application.
- GUI review console.
- Long-running cloud agent service.
- Replacing Git as the source of truth.
- Replacing Claude Code or Codex.
- Fully autonomous decision acceptance.
- Fine-tuning a model.
- Perfect semantic reproduction of a gold repo.
- Installing arbitrary private delegated repos without clear error reporting.
- Circumventing `mise-en-place` ownership/collision handling.

## Explicitly deferred

- MCP server.
- ChatGPT app/ChatKit wrapper.
- LangGraph-style durable orchestration service.
- Team dashboard.
- Enterprise policy server.
- Multi-user approval workflow.
